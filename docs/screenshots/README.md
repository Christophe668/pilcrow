# Screenshots

These are referenced by the main [README](../../README.md).

| File          | What it shows                                                    |
| ------------- | ---------------------------------------------------------------- |
| `library.png` | Web client, unread list with rail, filters, tags, and saves.     |
| `mobile.png`  | Android phone, unread list with filter pills, section, FAB, tab. |
| `kobo.png`    | KOReader plugin queue on a Kobo (status / tags / sort row).      |

To refresh:

- **Web:** open the deployed Pilcrow in a wide window (1200–1600px is
  a good range), Cmd+Shift+4 on macOS, save PNG over `library.png`.
- **Phone:** take a device screenshot from the unread library screen
  (Android: power + volume-down; iOS: side + volume-up). Save over
  `mobile.png`.
- **Kobo:** with the plugin installed and synced, take a device
  screenshot (KOReader: corner-tap → top-row icons → camera) or a
  Mac screenshot of the queue view; save over `kobo.png`.

Add more files here if you want extra slots in the README — e.g.
`reader.png` (an article open) or `highlights.png` (the global
Highlights view). The README will need its `<table>` extended to show
them.
